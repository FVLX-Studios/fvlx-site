# fvlx-site

The FVLX Studios website, hosted on GitHub Pages.

**Live site:** https://fvlxstudios.github.io/fvlx-site/

## Local development

It's a static site — just open `index.html` in a browser, or serve the folder:

```bash
python3 -m http.server
# then visit http://localhost:8000
```

## Deploying

Pushing to the `main` branch automatically updates the live site
(GitHub Pages is configured to deploy from `main` / root).
